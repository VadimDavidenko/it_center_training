1. Управление пользователями
-------------------------------
Написать приложение, позволяющее добавлять нового пользователя и просматривать список существующих пользователей.
Структура таблицы (id, имя, пароль, дата).

2. Аутентификация пользователей
----------------------------------
Написать приложение выполняющее аутентификацию пользователя на основе учётной записи созданнной в предыдущем задании.
Если пользователь зарегистрирован, то выводим список зарегистрированных пользователей,
если нет, выводим "Неправильный логин или пароль"
